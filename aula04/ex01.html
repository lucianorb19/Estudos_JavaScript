Mostrando na tela

Podem ser escritos com os comandos:

  • window.alert('Mensagem') - mostra algo na tela (alerta do navegador);
  • document.write('Mensagem') - mostra algo na tela (escreve no documento);
  • window.confirm('Mensagem') - mostra algo e o usuário precisa confirmar;

Entrada do usuário

  • window.prompt('Mensagem') - recebe uma entrada do usuário (string).

Variáveis JS

Os nomes de variáveis no JS são declaradas por var nome_variavel, seguindo o seguinte:

  • Podem começar com letra, $ ou _ ;
  • Não podem começar com número;
  • É possível usar acentos e símbolos;
  • Não podem conter espaços;
  • Não podem ser palavras reservadas;
  • Case Sensitive;

Tipos de variáveis:

  • boolean: true ou false;
  • null;
  • number - engloba inteiro, real (usar . e não ,), Infinity e NaN;
  • string - declarar com " ", ' ', ` `;
  • function;
  • symbol, object (array) e undefined ?

Pra identificar o tipo da variável, usar a função typeof.

Conversão de Tipos

String para Number

Tendo uma variável n string:

  • Number.parseInt(n) - converte n para inteiro;
  • Number.parseFloat(n) - converte para float;
  • Number(n) - a linguagem define se é Int ou Float.

Number para String

Dado um número n

  • String(n);
  • n.toString()

Formatação de Strings

Dada uma variável string nome

  • nome.length - tamanho da variável;
  • nome.toUpperCase() - torna todas letras maiúsculas;
  • nome.toLowerCase() - torna todas letras minúsculas;
  • nome.replace("x","y") - troca todos caractéres x por y.

Formatação de Números

Dada uma variável number num

  • num.toFixed(x) - deixa o número com x casas decimais
  • num.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) - formata o número para real brasileiro (memorizo nuuunca);
  • num.replace("x","y") - troca todos caractéres x por y (torna o número uma string)

Template String

Para facilitar a utilização de variáveis em frases, sem depender do uso de '' + '', usa-se ${} envolvendo a variável. Importante lembrar que só funciona se a frase estiver envelopada com crase.


        var nome = "Luciano"
        window.alert(`O nome do aluno é ${nome}`)

Operadores

Aritméticos

  • + soma ou concatenação (string)
  • - subtração;
  • * multiplicação;
  • / divisão;
  • % resto da divisão;
  • ** exponenciação;

Precedência de Operadores

  1. ()
  2. ** , sqrt
  3. / , * , %
  4. + , -

Atribuições:

  • x = x+2 --- x+=2
  • válido também para os demais operadores....
  • x = x+1 --- x++

    • Relacionais

    • > --- maior que;
    • < --- menor que;
    • >= --- maior ou igual;
    • <= --- menor ou igual;
    • == --- igual a (não compara tipo);
    • === --- igual a (compara tipo);
    • != --- diferente de.

    Dica:

    • 5 == "5" assume valor true
    • 5 === "5" assume valor false

    Lógicos (já com precedência)

    1. ! --- negação;
    2. && --- e;
    3. || --- ou

    Dica: Numa expressão com todos esses operadores, a precedência entre eles é: 1 - aritméticos; 2- relacionais; 3- Lógicos.

    Operador ternário

    É uma maneira simplificada de escrever um teste para uma expressão, que caso seja verdadeira, assume um valor, caso seja falsa, assume outro valor.

    teste ? true : false
