Herança de protótipo

A orientação a objetos do JS funciona baseada no conceito de herança de protótipos. Todos objetos do JS herdam de uma classe base, que aqui recebe o nome de protótipo, sendo que a base para todos os objetos é a classe Object. Logo, numa hierarquia de objetos, a classe que é usada como base é chamada de protótipo.

Para saber qual é o protótipo (classe mãe/ classe base) de um objeto, pelo console do navegador, podemos usar o comando nomeObjeto.__proto__.


    let user = {
    perfil: 'estudante'
    };

    let estudante1 = {
    nome: 'juliana'
    };

    Object.setPrototypeOf(estudante, user);
    estudante1.__proto__; // SAÍDA {perfil: 'estudante'}
